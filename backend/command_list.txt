<h2>Dev commands:</h2>
<ul>
    <li><code>"$sudo reset/rc"</code>: Resets chat rooms.</li>
    <li><code>"$sudo status"</code>: Gets the full status of the chat.</li>
    <li><code>"$sudo lines"</code>: Gets the line count of the chat room you are on.</li>
    <li><code>"$sudo pstats"</code>: Gives you partal status of the chat server.</li>
    <li><code>"$sudo clear_mutes"</code>: A debug command that clears all the mutes from every user (later you can pick one user or all).</li>
    
</ul>

<h2>Admin commands:</h2>
<ul>
    <li><code>"$sudo lock"</code>: Locks the chat.</li>
    <li><code>"$sudo unlock"</code>: Unlocks the chat.</li>
    <li><code>"$sudo mute (user) (time)"</code>: mutes user for your set time in the room you are in.</li>
    <li><code>"$sudo unmute (user)"</code>: unmutes the user in the chat in the room you are in.</li>
    <li><code>"$sudo ban (user) (time)"</code>: mutes the user in every room for a set amount of time.</li>
    <li><code>"$sudo reset"</code>: Resets chat rooms.</li>
    <li><code>"$sudo cmd_logs"</code>: Sends the last 10 entries in the command log.</li>
    <li><code>"$sudo globalock (confirm)"</code>: Locks all rooms. Only use in emergencies</li>
    <li><code>"$sudo word_ban (word)"</code>: Removes that word from the whitelisted words</li>
    <li><code>"$sudo word_unban (word)"</code>: Adds that word to the whitelisted words</li>
</ul>

<h2>User commands:</h2>
<ul>
    <li><code>"$sudo help</code>": You just used it so guess what this does now.</li>
    <li><code>"$sudo refresh</code>": Refreshes online list. (This is a useless command now, but still works.)</li>
    <li><code>"$sudo online</code>": Appear online.</li>
    <li><code>"$sudo offline</code>": Appear offline.</li>
    <li><code>"$sudo ping"</code>: Tells you how much ping you have.</li>
    <li><code>"$sudo create "room name"</code>: There was once a legendary command that let you own a room but was lost, but now its been found.</li>
</ul>
